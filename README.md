# Go Basic API

## Go, MySQL, Docker

### 🚀 Quick start

#### Para correr localmente se necesita correr la base de datos mongo con docker

```bash
docker-compose up -d
```

#### Start in development mode

```shell
npm run dev 
```

#### Borrar datos del entorno

```bash
docker-compose down -v
```
